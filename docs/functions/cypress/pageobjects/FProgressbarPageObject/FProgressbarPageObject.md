Använd `FProgressbarPageObject`

## Syntax

```ts
new FProgressbarPageObject(selector);
```

### Parametrar

`selector: string`
: Selector till `FProgressbar` elementet. Du kan med fördel använda {@link TestPlugin `v-test`} direktivet för din selector.
